# ThinkUnity

更多偏向UI的构建工程



---


Assects目录结构：

```
|-- Common                        /管理项目公共模块的变量目录
    |--Config                     /配置目录
|--Resources                      /资源目录
|--Scenes                         /场景目录
|--Scripts                        /脚本目录
    |--Controller                 /控制器目录
        |--Home                   /```Home```模块，项目默认模块  
    |--Model                      /模型。数据库相关操作。
    |--View                       /视图目录，存放对应的模版文件。如果支持国际化和多主题，那么视图目录下需要有对应的子目录。
        |--HomeView
            |--Config
```
